# Collections (Coleções)

Lista (implements List),
ordem é garantida, permite elementos duplicados, índice

lista.add 5
lista.add 9
lista.add 1
lista.add 5

for item : lista
  println item // 5, 9, 1, 5

Conjunto (implements Set)
ordem não é garantida, não permite elementos duplicados

conjunto.add 5
conjunto.add 9
conjunto.add 1
conjunto.add 5

for item : conjunto
  println item // 5, 9, 1
